<h1 align="center">
    Um Clone do YouTube construído com Django e React!
</h1>

<h2>Instalação</h2>

<h3>Backend</h3>
        
``` sh
$ cd backend
$ virtualenv venv
$ pip install -r requirements.txt
$ cd src
$ python manage.py migrate
$ python manage.py runserver
```        
<h3>Frontend</h3>
        
```sh
$ cd frontend
$ npm i
$ npm start
```
